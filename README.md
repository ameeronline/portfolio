# 🌐 ameeronline — Personal Portfolio Website

Welcome to the source code of my personal portfolio website — **ameeronline**. This site showcases my professional profile, projects, skills, experience, and contact details, built to reflect both modern design trends and best practices in web development.


## ✨ Features

- Fully **Responsive Design** (Mobile, Tablet, Desktop)
- Project Showcase with live demos and GitHub links
- Smooth page transitions and scroll animations
- Downloadable Resume / CV
- Contact Form integration
- SEO optimized with meta tags
- Deployed on GitHub Pages
